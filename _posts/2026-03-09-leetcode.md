# list 数据结构  
### 定义和初始化  
``` c++
#include <list> 
using namespace std; 
 
list<int> a;  
list<int> b = {1, 2, 3, 4};  
list<int> c(5);       // 5 个 0  
list<int> d(5, 7);    // 5 个 7  
```

### 头尾插入和删除  
``` c++
list<int> lst;

lst.push_back(10);
lst.push_back(20);
lst.push_front(5);

lst.pop_back();
lst.pop_front();
```

### 访问头尾  
``` c++
cout << lst.front() << endl;
cout << lst.back() << endl;
```

### 插入和删除指定位置  

``` c++
list<int> lst = {1, 2, 4};

auto it = lst.begin();
++it;  // 指向 2

lst.insert(it, 100);   // 在 2 前面插入 100

auto it2 = lst.begin();
++it2;   // 指向 100

lst.erase(it2);

```

### 大小和判空  
```
cout << lst.size() << endl;
cout << lst.empty() << endl;
```

### 这些函数是链表特别适合做的事。

- sort()
```
list<int> lst = {4, 2, 5, 1, 3};
lst.sort();
```

- reverse()
```
lst.reverse();
```
- remove(value)

 删除所有等于某个值的元素：
```
list<int> lst = {1, 2, 2, 3, 2};
lst.remove(2);   // 删除所有 2
```

 结果：
```
1 3
```

- unique()

删除连续重复元素：
```
list<int> lst = {1, 1, 2, 2, 2, 3, 1};
lst.unique();
```

结果：
```
1 2 3 1
```

注意它只去掉相邻重复，不是全局去重。
如果想全局去重，通常先排序再 unique()：
```
lst.sort();
lst.unique();
```

- merge()

 把两个有序链表合并：
```
list<int> a = {1, 3, 5};
list<int> b = {2, 4, 6};

a.merge(b);
```

 结果：
```
a 变成 1 2 3 4 5 6

b 变空
```
 前提是两个链表都已经有序。

- splice()

这是 list 很有代表性的操作：
可以把一个链表的节点直接接到另一个链表里，不需要拷贝元素。
```
list<int> a = {1, 2, 3};
list<int> b = {10, 20, 30};

auto it = a.begin();
++it;  // 指向 2

a.splice(it, b);
```

结果：
```
a 变成 1 10 20 30 2 3

b 变空
```
这就是链表的强项之一。

### 删除元素时的正确写法

很多人会在遍历时删除，结果把迭代器弄失效。

错误风险写法：
``` c++
for (auto it = lst.begin(); it != lst.end(); ++it) {
    if (*it == 2) {
        lst.erase(it);
    }
}
``` 

正确写法：
``` c++
for (auto it = lst.begin(); it != lst.end(); ) {
    if (*it == 2) {
        it = lst.erase(it);
    } else {
        ++it;
    }
}
```

因为：`erase(it)` 会删除当前节点  

并返回下一个有效迭代器

### 迭代器失效规则


 插入一个新节点，通常不会让其他迭代器失效 
 删除某个节点，只会让指向该节点的迭代器失效 

 其他节点的迭代器通常仍然有效

### 什么时候适合用 list

适合：

- 经常在中间插入/删除

- 已经拿到了要操作位置的迭代器

- 需要 splice、merge 这种链表特性

不太适合：

- 需要频繁按下标访问

- 需要高性能遍历

- 只是普通存数据、排序、查找


### 一个完整小例子
``` c++
#include <iostream>
#include <list>
using namespace std;

int main() {
    list<int> lst = {3, 1, 4, 1, 5};

    lst.push_front(0);
    lst.push_back(9);

    cout << "初始: ";
    for (int x : lst) {
        cout << x << " ";
    }
    cout << "\n";

    auto it = lst.begin();
    ++it; // 指向 3
    lst.insert(it, 100);

    cout << "插入后: ";
    for (int x : lst) {
        cout << x << " ";
    }
    cout << "\n";

    lst.remove(1);

    cout << "删除 1 后: ";
    for (int x : lst) {
        cout << x << " ";
    }
    cout << "\n";

    lst.sort();

    cout << "排序后: ";
    for (int x : lst) {
        cout << x << " ";
    }
    cout << "\n";

    lst.reverse();

    cout << "反转后: ";
    for (int x : lst) {
        cout << x << " ";
    }
    cout << "\n";

    return 0;
}
```
