# vector相关 整行赋值的办法  

```
vector<int> v ;  
v = {1,2,3,4};  
v.assign(5,9); //{9,9,9,9,9}  
vector<vector<int>> m;  
m[1] = {7,8,9,10};  
m[2] = vector<int>(4,5);  
```

# Typedef 的使用  
```
typedef unsigned long long ULL;  // 简化长类型  
typedef int* IntPtr;             // 指针别名  
typedef int Arr5[5];             // 数组类型别名  
typedef int (*Func)(int, int);   // 函数指针别名  
```
